# BK7252N OpenCam — план порта и статус

Цель: локальный MJPEG-поток в браузер/VLC по LAN, без китайского облака.
База: `daniel-dona/beken7252-opencam` (RT-Thread + Beken BDK), портируется на BK7252N.

## Архитектура решения

```
  [сенсор GC0311/…]
        | 8-bit DVP
        v
  [DVP/CIS]→[JPEG энкодер]→[GDMA]→ RAM (буфер кадра)      <-- ДРАЙВЕР (chip-specific, BK7252N)
        |
        | video_transfer_init(send_func=mjpeg_send)        <-- opencam API (есть)
        v
  [mjpeg_http.c] --HTTP multipart/x-mixed-replace--> браузер/VLC   <-- НОВОЕ (chip-independent)
  [net_cfg.c]    --Wi-Fi STA (роутер) + SoftAP fallback "opencambk"  <-- НОВОЕ
  [EasyFlash]    --хранит SSID/pass, меняется на лету без пересборки  <-- как в стоке
```

## Что в этой папке (chip-independent, готово к интеграции)

| Файл | Назначение | Статус |
|---|---|---|
| `net_cfg.h/.c` | Wi-Fi STA + SoftAP fallback, конфиг через EasyFlash, дефолт зашит | написано, требует сборки |
| `mjpeg_http.h/.c` | HTTP-сервер :80, отдаёт `multipart/x-mixed-replace`; кадры режет по JPEG-маркеру FFD8 | написано, требует сборки |
| `app_entry.c` | точка входа: монтирование, сеть, авт0определение сенсора, старт сервера | написано |
| `bk7252n_cam_regs.h` | карта регистров камеры BK7252N (из реверса), с пометками уверенности | каркас, см. ⚠ |

## Что ещё нужно (chip-specific, НЕ в этой папке)

**Драйвер камеры под BK7252N** — единственный недостающий блок. Опции:
- **Маршрут A:** взять `camera_intf.c`/`jpeg`/`i2c` из Beken SDK **3.0.76+** (там адреса `0xA0xxxx`
  уже верные) — подставить вместо opencam-овских. Самый быстрый и надёжный.
- **Маршрут B:** доковырять битовые поля DVP/JPEG/I2C из стока по адресам из `bk7252n_cam_regs.h`
  (JPEG-блок `0xA03000`, enable `0xA03034` бит `0x10` — уже найдено).

См. `../RE_findings_bk7252n.md` — там карта регистров и оговорки по уверенности.

## Конфиг сети (зашитый дефолт)

- SoftAP SSID: `opencambk`  пароль: `imeXScdxADaUHuUU`  (IP `192.168.169.1`, поток `http://192.168.169.1/`)
- STA: задаётся в EasyFlash ключами `sta_ssid`/`sta_pass` (или меняется командой в shell); если
  не задан или не подключился за 15 c — поднимается SoftAP `opencambk`.
